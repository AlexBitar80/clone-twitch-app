Este projeto foi desenvolvido utilizando as senguintes tecnologias:

-  [TypeScript](https://www.typescriptlang.org/)
-  [React-Native](https://reactnative.dev/)
-  [Expo](https://expo.io/)
-  [React Navigation](https://reactnavigation.org/)
-  [styled-components](https://styled-components.com/)

 
## :information_source: Como rodar

 para clonar este repositório e rodar o projeto você deve ter instalado em sua máquina o [Expo](https://expo.io/) o [Git](https://git-scm.com/) e o [Yarn](https://yarnpkg.com/) 

```bash
# Clone este repositório em uma pasta de sua preferência
$ git clone https://github.com/AlexBitar80/clone-twitch-app.git

# navegue até a pasta 
$ cd (pasta que foi clonada)

# instale as dependências
$ yarn install

# rode o projeto
$ yarn start
```

o projeto irá abrir uma página em seu navegador, basta instalar o app do expo em seu Android ou IOS e escanear o QR CODE ou escolher a opção de rodar em seu emulador
